# Gym Companion

A full-stack platform for personalized fitness guidance powered by AI.

## Monorepo Structure

```
gym-companion/
├── client/          # React + Vite + Tailwind CSS frontend
├── server/          # Node.js + Express REST API
├── ai-service/      # Python FastAPI AI microservice
└── README.md
```

## Services

### Client
React single-page application with Vite and Tailwind CSS.

### Server
Node.js REST API backend using Express.

### AI Service
Python FastAPI microservice for AI-powered posture and fitness recommendations.

## Getting Started

1. Clone the repository
2. Install dependencies for each service
3. Start the services
